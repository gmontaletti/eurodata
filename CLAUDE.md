## Project Overview

**eurodata** is an R package that collects regional (NUTS-2) indicators from Eurostat for EU27 countries, transforms them into a unified schema, selects a peer group of comparable regions, and exports the results. The package focuses on labour market, education, demographics, ICT, and economic indicators.

Author: Giampaolo Montaletti (giampaolo.montaletti@gmail.com)

## Exported API

| Function | Purpose |
|----------|---------|
| `ed_collect(dataset_ids, nuts_year, countries)` | Download Eurostat datasets, filter to valid NUTS-2 regions |
| `ed_transform(data_list, specs)` | Apply spec-driven transformations, produce unified data.table |
| `ed_peergroup(dt, reference_region, ...)` | Select peer regions comparable to a reference (default: Veneto) |
| `ed_export(dt, path, format)` | Write results to RDS and/or XLSX |
| `dataset_specs` | List-of-lists defining all 24 dataset transformations |

## Key Libraries

- `eurostat` -- API access to Eurostat bulk data
- `data.table` -- data manipulation, rbindlist, dcast
- `dplyr` -- filtering and reshaping within individual datasets
- `regions` -- NUTS region validation (recode_nuts, validate_nuts_regions)
- `writexl` -- Excel export (suggested)

## Conventions

- Section comments in R scripts use `# 1. name of section -----` syntax (no `####`-only rows)
- Internal helpers are dot-prefixed (e.g., `.apply_spec()`, `.apply_filters()`)
- Transformation logic is configuration-driven via `dataset_specs` in `R/config.R`
- Indicator codes (`iCode`) are short mnemonics (e.g., `PPSprcptP`, `OccTass`, `DisLungaP`)
- The peer group is selected by comparing regions to Veneto on population and PPS

## Data Flow

```
Eurostat API
     |
     v

     |
     v
ed_transform(data_list, dataset_specs)  -->  unified data.table
     |                                        (iCode, iName, uCode, uName, TIME_PERIOD, values)
     v
ed_peergroup(dt)  -->  filtered peer group data.table
     |
     v
ed_export(dt, path, format)  -->  .rds / .xlsx files
```
